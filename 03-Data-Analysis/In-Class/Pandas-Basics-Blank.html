Import Pandas¬∂

To use the Pandas library, we first need to import it.

import pandas as pd

The above import statement not only imports the Pandas library but also gives it an alias or nickname ‚Äî pd. This alias will save us from having to type out the entire words pandas each time we need to use it. Many Python libraries have commonly used aliases like pd.

By default, Pandas will display 60 rows and 20 columns. I often change Pandas‚Äô default display settings to show more rows or columns.

pd.options.display.max_rows = 200
